# Prompt Engineering Comparison: Zero-Shot vs Few-Shot vs Chain-of-Thought

## Overview

This project explores how different prompting techniques influence the responses generated by a Large Language Model (LLM) for sentiment classification tasks.

The objective is to compare the effectiveness of three widely used prompting strategies:

* Zero-Shot Prompting
* Few-Shot Prompting
* Chain-of-Thought (CoT) Prompting

By running the same input through different prompt structures, we can observe how prompt design impacts model behavior, consistency, reasoning, and output quality.

---

## Prompting Techniques

### Zero-Shot Prompting

Zero-shot prompting asks the model to perform a task without providing any examples.

**Characteristics**

* Minimal prompt design effort
* Fast and token-efficient
* Suitable for straightforward tasks
* May produce inconsistent output formats

---

### Few-Shot Prompting

Few-shot prompting provides a small set of examples that demonstrate the expected task and output format before presenting the actual input.

**Characteristics**

* Improves output consistency
* Helps guide model behavior
* Useful when specific response formats are required
* Requires additional prompt tokens

---

### Chain-of-Thought Prompting

Chain-of-Thought prompting encourages the model to reason through the problem before producing a final answer.

**Characteristics**

* Improves explainability and transparency
* Useful for complex or ambiguous tasks
* Helps understand how the model arrives at a conclusion
* Typically generates longer responses

## How It Works

1. Define a sentiment classification task.
2. Create prompts using different prompting strategies.
3. Send prompts to an LLM through an API.
4. Collect model responses.
5. Compare the outputs generated by each prompting technique.

## Key Concepts Demonstrated

### Prompt Engineering

The practice of designing effective prompts to guide LLM behavior and improve response quality.

### Sentiment Classification

A Natural Language Processing (NLP) task that identifies whether a piece of text expresses:

* Positive sentiment
* Negative sentiment
* Neutral sentiment

### Reasoning vs Direct Prediction

The project highlights the difference between:

* Direct classification (Zero-Shot)
* Example-guided classification (Few-Shot)
* Step-by-step reasoning (Chain-of-Thought)
